## Description

Car Dealer Application

## Technologies

- Node.js
- Nest.js
- PostgreSQL
- TypeORM
- Redis
- Docker
- Cypress

## Requirements

- Docker
- Docker Compose
- node (v22)
- npm
- npx

## Installation

```bash
$ npm install
```

## Running the app

```bash
# docker services
$ npm run start:docker:local

# api server
$ npm run start:local

# reset and seed initial data
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
